# array  
## 定义
```
int num[100]; //定义了一个名为num的数组，可以储存100个int的量  
double ages[20];
int num[3][5]; //多维数组 三行五列
```
### 数组的特点
* 数组一旦被定义，大小就不能改变
* 数组在内存里是连续储存的
* 数组里的类型必须相同
### 数组越界
```
int num[2];
//num的有效索引为num[0],num[1]
//num[2]越界
```
编译器是不会查数组是否越界，所以有时候即使越界也不报错，但在不同的电脑上有可能报错，所以如果出现难以解释的情况，比如代码在一个电脑上可以跑，在其他地方不能跑，很有可能是越界导致的，所以检查数组是否越界是程序员的责任。
## 数组的初始化
```
int num[10];
for (int i = 0;i<10;i++)
{
    num[i]=0;
}

int num[3]={1,2,3};
int num[]={4,5,6};
int num[2][3]={{1,2,3},
               {4,5,6}};
```
