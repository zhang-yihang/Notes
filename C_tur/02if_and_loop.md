# if 语句  
## 语法结构  
```
if (a>10)
{
    printf("a>10");
}
else if(a>3)
{
    printf("a>3">)
}
else
{
    printf("a<=3")
}
```
## 条件判断  
|  符号   | 含义  |  
| :----: | :----: |     
| ==  | 等于 |   
| !=  | 不等于 |  
|   &lt; = | 下雨等于 |  
| &gt; = | 大于等于  |  
| &gt; | 大于  |  
|&lt; | 小于 |
